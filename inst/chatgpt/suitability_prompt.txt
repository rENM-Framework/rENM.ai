AREA COMPUTATION RULES (km²)
-----------------------------------------------------------------------
1. Reproject rasters to an equal‑area CRS (EPSG:6933).
2. Compute per‑cell area:
   cell_area_km2 = |a11 * a22| / 1,000,000
3. Modeled extent (km²) = non‑NA cell count × cell_area_km2.
4. Category areas (km²) = cell count in category × cell_area_km2.
5. Do NOT report “0.00 km²” unless the category truly contains zero cells.

Trend classification thresholds (for both rasters):
- increasing trend: value ≥ +0.0001
- stable (near zero): −0.0001 < value < +0.0001
- decreasing trend: value ≤ −0.0001

These thresholds also define “positive,” “negative,” and “stable” areas.
